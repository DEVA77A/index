Mern Stack

MERN Stack = MongoDB + Express.js + React.js + Node.js
Use: Full-stack JavaScript framework for building web apps.
MongoDB: Database
Express.js: Backend framework
React.js: Frontend UI
Node.js: Server runtime
Flow: React → Express/Node → MongoDB → back to React
